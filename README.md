# Movie Search
---
A simple movie discovery web application that allows users to search movies, browse by category, and view detailed movie information.

## Features

* Search movies by title, genre, director, or cast
* Browse movies by category
* View movie ratings and release years
* See detailed movie information in a popup modal
* Real-time search results
* Responsive design for desktop and mobile
* Built-in movie database (no API required)

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome
* Google Fonts

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/movie-search.git
```

2. Open the project folder:

```bash
cd movie-search
```

3. Open `index.html` in your browser.

## Usage

### Search Movies

* Type in the search bar.
* Results update automatically as you type.

### Browse Categories

* Select a category from the dropdown menu.
* Click the browse button to view movies in that category.

### View Details

* Click on a movie card.
* A popup window will display detailed information about the movie.
